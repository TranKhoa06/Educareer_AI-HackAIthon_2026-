\# Contributing to EduCareer AI



Thank you for your interest in contributing to EduCareer AI.



EduCareer AI welcomes contributions from developers, students, educators, career counselors, researchers, designers, and members of the open-source community.



\## Ways to Contribute



You can contribute by:



\- Reporting bugs.

\- Suggesting new features.

\- Improving documentation.

\- Improving accessibility.

\- Adding automated tests.

\- Improving the user interface and user experience.

\- Reviewing Vietnamese educational content.

\- Adding or reviewing career information.

\- Developing new career simulations.

\- Improving privacy, security, and responsible-AI practices.



\## Before Contributing



Please:



1\. Search existing issues before creating a new issue.

2\. Create an issue before making major changes.

3\. Do not publish private student information.

4\. Do not commit API keys, access tokens, passwords, or secret credentials.

5\. Use clear commit messages.

6\. Test your changes before submitting a pull request.
